# Pet-project Chess

### Функционал:
#### Реализовано:
* старт по готовности - нажатие на кнопку
* остановка партии - сброс доски
* очередность хода
* подсветка доступных ходов
* проверка первого хода пешкой
* движение фигур
* атака только фигур соперника
* корзина "побежденных" фигур соперника
